# Performance

Refer to the [project README](https://github.com/fraware/TinyRL/blob/main/README.md#performance) for indicative tables (CartPole, Acrobot, Pendulum-style environments). Reproduce baselines with `./reproduce.sh` when present on your platform.

CI runs **short** training smokes to ensure configs still execute; they are not authoritative benchmarks. Use `benchmark_harness.py` locally for comparisons.
